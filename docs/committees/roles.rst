.. note::
   - **People（人）**：人の関係性・チームワーク・モチベーションに関わる要素
   - **Process（プロセス）**：仕組み・ルール・運用フローの設計・改善
   - **Technology（技術）**：IT・ツール・システムによる支援・高度化

各係の活動は、大きく以下の2つに分けて考えます。

- **日常業務**：研究室を円滑に回すための日常業務・タスク
- **改善・改革**：現状の運営や仕組みに対する課題発見、より良くするための提案・実行

特に、改善活動は、経営工学のコア中のコアのスキルとなります。係は単なる「作業担当」ではなく、改善を回し続ける主体であることを期待しています。各自が「もっと良くできるのでは？」と考え続けてください。その積み重ねが、研究室全体のレベルを引き上げることに繋がります。
