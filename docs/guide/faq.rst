よくある質問（FAQ）
===================

.. note::
   随時追加予定です。気になることがあれば大森に連絡してください。

ゼミ・スケジュール
------------------

**Q. ゼミを欠席する場合はどうすればいいですか？**

事前に大森にメールで連絡してください。やむを得ない欠席の場合は、ゼミ開始までに資料をアップロードすることで0.5回分の出席として扱われます（学部生）。

**Q. 個別ミーティングのスロットはどこで確認しますか？**

:doc:`../resources/links` の「Zoom アドレス・ミーティング一覧」スプレッドシートを参照してください。

**Q. 海外学会や国内学会への参加費・旅費はどうなりますか？**

標準的な国内学会（日本経営工学会全国大会）の参加費は実験実習費から支出します。旅費が高額になる海外学会については別途相談してください。

論文・研究
----------

**Q. 論文の引用スタイルは何を使えばいいですか？**

APAスタイルで統一してください。Google Scholar の「引用」ボタンから取得できます。

**Q. 実データを企業から入手したい場合はどうすればいいですか？**

NDA（秘密保持契約）の締結が必要で、平均1〜2ヶ月かかります。早めに大森に相談してください。なお、企業側に協力するメリットがある提案（面白い仮説・実用的なメリット）が前提条件となります。

**Q. プログラミング言語は何を使えばいいですか？**

特に指定はありませんが、研究室では Python・Julia・R を主に使用しています。再現性のためにシード固定・パラメータの先頭まとめを徹底してください。詳しくは大森に相談してください。

コミュニケーション
------------------

**Q. 大森との1on1ミーティングはどう申し込みますか？**

カレンダーの空きスロットを確認して、メールで申し込んでください。修士学生のミーティングスロットの目安はゼミシラバスに記載しています。

**Q. フィードバックが厳しいと感じた場合は？**

フィードバックは「教員の意見」であり、最終的な意思決定は本人に委ねられています。改善のヒントとして受け取ってください。違和感がある場合は遠慮なく伝えてください。
