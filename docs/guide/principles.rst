研究室プリンシプル
==================

大森研究室のメンバーが日々の研究・活動において大切にする考え方です。

.. raw:: html

   <div class="principles-list">

   <div class="principle">
     <h3>良い研究室は全員で作り上げるもの</h3>
     <p>良い研究室は、教員だけではなく、学生が作り上げるものであるという信念のもと、本研究室は運営されています。研究室で仲間と過ごす時間は、学生生活の大きな部分を占め、卒業後も一生の財産になります。そのような時間をより良いものにするために、自分たちで良い環境を作ることが重要です。自らコミュニティに貢献できることを見つけるとともに、コミュニティからも協力を得ながら、互いに成長し合える環境を皆で作りましょう。</p>
   </div>

   <div class="principle">
     <h3>「価値の消費者」から「価値の生産者」へ</h3>
     <p>授業では授業料を払って知識を得ていますが、大学を卒業・修了した後には「価値を生み出せる人材」であることが求められます。</p>
     <ul>
       <li><strong>価値の消費者</strong>：お金を払ってサービスを受ける</li>
       <li><strong>価値の生産者</strong>：お金をもらえるサービスを提供できる</li>
     </ul>
     <p>価値とは「それを求める人」がいて、初めて対価としてお金が支払われます。そして、「相手がどれだけそれを求めているか」により、その価値の大きさが決まります。したがって、研究においても研究室生活においても、相手のニーズを理解し、それを満たす解決策を持つ「価値の生産者」であることに拘りましょう。</p>
   </div>

   <div class="principle">
     <h3>経験から学ぶ（Learn by Doing）、他者から学ぶ（Learn from Others）</h3>
     <p>
       今では、多くの情報がインターネット・AIを活用して独学することができる様になっています。
       この様な中で、大学で学ぶ意義として以下の2つが特に重要になります。
     </p>
     <ul>
       <li><strong>Learn by Doing</strong>：試行錯誤の経験を通して学ぶ</li>
       <li><strong>Learn from Others</strong>：他人から学ぶ</li>
     </ul>
     <p>
       学びの質は、自分自身の経験の質や、自分の周りにいる他人（ラーニング・コミュニティ）の質に強く依存します。
       高い基準を持ち、学びの相乗効果が最大限に働き、成長できる環境を作りましょう。
       これに加えて、外部の人とも学び合える関係を築けるよう、魅力ある研究室を全員で作っていきましょう。
     </p>
   </div>

   <div class="principle">
     <h3>高い基準と高い心理的安全性</h3>
     <p>高い基準（High Standards）と心理的安全性（Psychological Safety）は、両立させることが重要です。心理的安全性が低い環境では、失敗を恐れて挑戦や率直な意見が生まれません。一方で、基準が低い環境では、馴れ合いとなり成長が止まります。互いを尊重しながら率直に意見を言い合い、高い基準で挑戦し続ける環境を作りましょう。</p>
   </div>

   <div class="principle">
     <h3>フィードバックはギフト</h3>
     <p>フィードバックの原則は、「フィードバックは自分の意見であり、意思決定は相手に委ねる」ということです。伝える側は、相手が受け取りやすいタイミング・言い方を心がけたうえで、相手の成長を考えて率直に伝えることが重要です。受ける側は、「自分 vs 相手」ではなく、「自分・相手 vs 問題」という意識で捉え、必要だと判断したものを積極的に取り入れたうえ、感謝の意を伝えましょう。</p>
   </div>

   <div class="principle">
     <h3>ショートカットはない</h3>
     <p>近道はありません。近道をしようとすると、自分に返ってきます。<br>自分と向き合い、努力を積み重ねましょう。</p>
   </div>
