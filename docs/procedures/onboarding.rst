   * - 項目

     - 鍵番号を記録・記憶して一人で研究室に出入りできるようになってください。
   * - ロッカー
     - ロッカーを割り当ててもらってください（B4以上）。

1on1ミーティング
-----------------

各学年1人以上にアポイントをとってミーティングをして頂くことを推奨しています。ミーティングでは以下の様なことを聞いて頂ければと思います。

- ご自身の自己紹介・研究内容を紹介する。
- 対象学生の研究内容・各学年の雰囲気・授業の履修アドバイス等を聞く。

いきなり連絡を取ることは少し心理的なハードルが高いと思いますが、（外資系などで）転職時に、既にその組織にいる人に1on1を申し込んで関係作りをする文化がありますので、その様な練習だと思って頂ければと思います。

全員、この様なことには慣れているので、突然連絡をもらったとしても快く対応してくれると思います。

どなたを選んで頂いても構いませんが、選びにくければ大森やその他の知っている学生にご相談頂ければと思います。
