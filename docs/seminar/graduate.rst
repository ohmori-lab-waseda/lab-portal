修士ゼミ（M1・M2）
===================

2026年度 大森研究室ゼミ・シラバス

----

１．授業の概要
--------------

修士論文作成のための学習を行う。研究内容に関しては、学部編に準じる。原則的な考え方として、修士課程では「専門家として扱う」こと重視する。研究に関しては、プロセスに関してはより自由がある中で自身のプロジェクトをマネジメントし、研究の成果を出すことを学ぶ。これに加えて、より指導的な立場としての言動を身に着ける。

**(1)「知の受容者」から「知の生産者」へ**

授業では「知の受容者」として知識を得ることが中心だが、研究では「知の生産者」となることが求められる。学部の卒業論文では、探索的にさまざまなことに挑戦し、学ぶ姿勢を養うことも重要であるが、「学ぶだけ」では新たな価値を生み出したことにはならない。そのため、修士課程では結果にこだわり、実際に成果を生み出すことが、最も大きな成長につながる。単に学部と同じようなことをもう一度経験するだけでは、成長の機会としては限定的になる。

修士課程修了後、多くの方は社会人になるが、社会では「価値を生産できるかどうか」が最も重要視される。

- 「価値の消費者」：お金を払ってサービスを受ける
- 「価値の生産者」：お金をもらえるサービスを提供できる

大学では「知」がプロダクトとなるため、新たな知識を生み出すことが価値の創造につながる。そして、どの業界・企業においても「価値を生み出せる人材」は常に求められている。研究を通じて成果を出すためには、論理的思考力・コミュニケーション能力・プロジェクトマネジメント力・データ分析力など、さまざまなスキルが求められる。これらを磨くことは、研究の成功だけでなく、就職活動や今後のキャリアにもプラスに働く。成果が出せれば、それもアピール材料になる。

**(2)「研究成果」を発表・出版することを目標に**

研究成果は、世の中に発信して初めて「知識」として認められる。どれほど素晴らしい内容であっても、発表・出版されなければ、世の中にとっては存在しないのと同じと言われている。成果のクオリティの指標として、以下のような順序で価値が高い。

- 査読付き論文（最も厳しい審査を通過した研究成果）
- 査読付き国際会議（海外学会）
- 国内学会
- 学内の修士論文発表会

このような高い目標にチャレンジすること自体が、大きな成長機会になる。

**(3)「研究プロセス」は各自に任せる**

上記の目標が明確であれば、研究の進め方は各自の裁量に任せる。各学生は、既に良い大人で、信頼しているため、必要なタイミングで適切にサポートを求めることができると考えている。ただし、国内学会発表などの中間ゴールを設定しており、そこで成果が出せない場合は、挽回の機会を増やせるようフィードバックを強化する。つまり、早めに努力すればするほど、その後の自由度を確保しやすい設計になっている。

２．授業の到達目標
------------------

**(1) 研究成果に関する到達目標**

成果物の到達目標としては、研究成果を発信することが求められる。成果の発信場所としては、①査読付き論文、②査読付き国際会議、③国内学会の順番で評価が高くなる。奨学金の免除を希望する場合にも、この順番で免除確率が高くなる。

① 投稿してから審査に半年〜9か月ほどの時間がかかるため、早めに研究を進めることが求められる。博士課程では査読付き論文2本が論文着手条件となるため、将来的に検討している学生は修士の間に1本の論文投稿をすることを薦める。

② 査読付き国際会議を標準的なものとする。発表時期はM2の10,11月であり、英語によるFull Paper提出が8月である。このため、M2の6月頃には研究結果が出ている必要がある。また、特に優れた研究に関しては、欧米などの国際会議を提案する場合がある（基準としては、①のレベルを想定している）。また、旅費が高額になるが、財源は実験実習費以外から支出することを想定している。

③ 日本経営工学会全国大会を標準的なものとする。毎年、春季大会（関東・5,6月）と秋季大会（地方・10,11月）の年2回、開催される。研究の進捗に応じて、M1の秋・M2春・M2秋などに発表することができる。

**(2) 指導的な立場としての到達目標**

**① ロールモデルとしての言動**

最も重要なことの1つが、自分自身が良いロールモデル（お手本）になることである。自分自身が相手にとって欲しいと思う行動を実践していることが重要である。その様にしていれば、自然と学部生からの尊敬を得ることができ、同じような行動を真似する様になる。反対に、自分自身がやれていないことを、学部生に押し付ける様な行動が続くと、次第に尊敬が失われてしまうため注意が必要である（例：遅刻しない、提出期限を守る、研究室に貢献する、掃除する）。

**② 課題設定**

研究においては、プロジェクト管理をしっかりとできる様に、課題や納期を設定し、定期的に進捗を確認する。また、自分自身がしっかりと研究・仕事を行っているうえで、全ての仕事を修士がやっていると、学部生にとっての成長機会を奪うことになるため、日常の仕事も可能な限り、学部生に覚えてもらう方が良い。やって見せて、やってもらい、フォローする様にする。

**③ 人間関係構築**

フラットな関係で接し、相手の心理的安全性を高め、率直な意見を言い合える様な関係を気づくことが重要である。適切な問いかけや、傾聴を行うことにより、本人が自分自身で問題に気付き、問題を整理することも非常に有効である。

**④ フィードバック**

フィードバックの原則としては、「フィードバックは自分の意見であり、意思決定は相手に委ねる」ということをお互いの間で共有しておくことである。相手が受け取りやすいタイミングで、受け取りやすい言い方を心掛けたうえで、レベルの高い研究や仕事ができるために改善すべき点は遠慮せずに率直に言うことが重要である。また、重要なことは何度も何度も繰り返すことは、非常に有効である。

**⑤ エスカレーション**

対象学生の状況を把握し、問題が発生する兆しがあった場合は、適切なタイミングでエスカレーション（指導教員などに報告・相談）し、必要なサポートを受けられる様にすることは重要である。問題とは、精神的不調・ハラスメント行為（同学年・親子・教員など全ての関係で起き得る）・人間関係などが挙げられる。

.. list-table::
   :header-rows: 1
   :widths: 28 72

   * - 能力
     - 到達目標
   * - ロールモデルとしての言動
     - | ✓ 自分自身が良いロールモデル（お手本）になることができる。
       | ✓ 指導対象の学部生からの尊敬を得ることができる。
   * - 人間関係構築
     - | ✓ 相手の心理的安全性が高い状態を作り、率直な意見を言い合える関係を築くことができる。
       | ✓ 相手のモチベーションを上げることができる。
   * - プロジェクトマネジメント
     - | ✓ 相手の状況を傾聴し、目標設定・日程計画・進捗管理ができる。
       | ✓ 本人・研究室・学科が望む目標達成をサポートできる。
   * - フィードバック
     - ✓ 効果的なフィードバックを行うことができる。
   * - エスカレーション
     - ✓ 適切なタイミングでエスカレーションし、必要なサポートを得ることができる。

６．成績評価方法
----------------

**(1) 修士論文**

研究成果（Product）と研究過程（Process）の両面から評価を行う。特に、研究成果に関する目安としては以下の通りである。

.. list-table::
   :header-rows: 1
   :widths: 15 85

   * - 成績
     - 成果発信
   * - A+
     - | ✓ 査読付論文として掲載：十分条件
       | ✓ 国際会議での発表：必要条件
   * - A
     - | ✓ 国際会議での発表
       | ✓ 国内学会での発表
   * - B,C
     - ✓ 成果発信なし

**(2) 生産・サプライチェーンマネジメント研究演習**

以下の観点から指導教員が評価する。

- 研究の進捗度合い
- 日常の研究室内での貢献度合い
- 出席点

**(3) 出席ルール**

修士の相談スロットについては、以下の回数を基準とする。但し、出席は取らないため、各自でセルフチェックする。

.. list-table::
   :header-rows: 1
   :widths: 15 42 43

   * - Q
     - M1
     - M2
   * - 春Q
     - 毎週15分の1on1
     - Qに3回以上
   * - 夏Q
     - Qに3回以上
     - Qに3回以上（国内会議出せなかった学生は毎週）
   * - 秋Q
     - Qに3回以上
     - Qに3回以上（国際会議出せなかった学生は毎週）
   * - 冬Q
     - Qに3回以上
     - 都度設定

学部の相談スロットについては、修士学生も出席を取り、成績の参考にする。特に、指導的立場として、恥ずかしくない行動をとり、学部生に比べて明らかに出席が少ない、遅刻が多いことがない様にする。また、欠席しないことが前提ではあるが、「就職活動のため」だけではなく、予め設定されたスロットである等のエビデンスを事前報告する。
